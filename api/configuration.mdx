<ParamField path="/data/index.bleve" type="path">

<ParamField path="/data/vectors.bin" type="path">

<Expandable title="Embedding Process Details">

### Normalization

All vectors are L2-normalized before storage:

```go internal/embed/embed.go:32-47
norm = sqrt(sum(v^2 for v in vec))
for i in range(len(vec)):
    vec[i] = vec[i] / norm
```

This allows cosine similarity to be computed as a simple dot product:
```
cosine_sim(a, b) = dot(a, b)  // when both are normalized
```

### Request Format

```json
{
  "model": "all-minilm",
  "prompt": "text to embed"
}
```

### Response Format

```json
{
  "embedding": [0.123, -0.456, ...] // 384 float64 values
}
```
